# 交通事故赔偿计算器

基于中国现行法律法规的交通事故人身损害赔偿计算工具。

## 快速开始

双击 `index.html` 文件，在浏览器中打开即可使用。无需安装任何软件。

## 功能特点

- 🧮 **完整赔偿计算** — 覆盖 14+ 项赔偿项目
  - 医疗费、后续治疗费、康复费
  - 误工费、护理费
  - 交通费、住宿费
  - 住院伙食补助费、营养费
  - 残疾赔偿金 / 死亡赔偿金
  - 丧葬费
  - 被扶养人生活费
  - 精神损害抚慰金
  - 残疾辅助器具费、鉴定费、财产损失

- 📊 **智能计算引擎**
  - 自动年龄分段（≤60 / 60-75 / ≥75）
  - 伤残等级系数自动换算（1级100% ~ 10级10%）
  - 被扶养人生活费单独计算
  - 责任比例自动折算
  - 交强险已赔付金额扣减

- 📄 **双格式导出**
  - Word (.docx) — 正式赔偿计算书
  - Excel (.xlsx) — 分项明细表

- 🗺️ **内置 31 省份 2025 年度赔偿标准**
  - 城镇居民人均可支配收入
  - 城镇居民人均消费支出
  - 城镇非私营单位在岗职工平均工资
  - 住院伙食补助费标准

## 法律依据

- 《中华人民共和国民法典》第一千一百七十九条
- 《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》
- 各省、自治区、直辖市统计局公布的 2025 年度统计数据

## 免责声明

本工具计算结果仅供参考，不构成法律意见。实际赔偿金额以法院判决或双方协商为准。

## 系统要求

- 现代浏览器（Chrome、Edge、Firefox、Safari）
- 首次使用需要网络连接加载 CDN 库（后续可离线使用）
